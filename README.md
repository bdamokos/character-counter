# Character Counter

## Live Site

**GitHub Pages:** <https://bdamokos.github.io/character-counter/>

[![Character Counter screenshot](./assets/character-counter-screenshot.png)](https://bdamokos.github.io/character-counter/)

Single-page HTML app that recreates the useful part of the Instagram counter workflow from
`charactercounter.com/instagram` without ads or filler.

## Run Locally

Open `index.html` directly in a browser, or serve the directory with any static file server.
